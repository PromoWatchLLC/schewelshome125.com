1. HOW TO ENTER: Beginning 8/24/21 through 10/31/21 (“Promotion Period”), visit a participating Schewels Home

from a Schewels Home store representative at the conclusion of your Test-Rest experience. Each Game Piece will
contain a ten (10) character unique code (“Code”). The Code will be printed in two (2) locations on the Game
Piece. The Schewels Home store representative will provide each entrant with a portion of the Game Piece
containing the Code for the entrant to retain. The other portion of the Game Piece will be retained by the
participating store and entered into the Sweepstakes. At the conclusion of the Promotion Period, one (1) $1,000
winning Code will be randomly drawn for each of the fifty (50) participating stores from all eligible entries received
for the respective store. Additionally, all eligible entries received for all participating stores will be included into a
random drawing to select one (1) $10,000 Grand Prize-winning Code from all eligible entries received for the
Promotion Period. Entrants will then check the Schewels Home website at www.schewelshometestrest.com or visit
a participating Schewels Home store to check to see if the randomly selected Codes match the Code on their Game
Piece. If an entrant possesses a Game Piece that contains a winning Code, he/she must visit the participating store
where they received their Game Piece in order to claim their prize. Further details on the random drawing, prize
claim process, prizes and other promotion information is described below. Limit one (1) Game Piece per person,

is illegible, mutilated, altered, duplicated, tampered with, forged, cut, torn, contains printing, typographical or other
